Web & HTML Basics

Introduction to the Web

The world wide web is a collection of Hyper Text Markup Language (HTML) documents that are connected by hyperlinks. Various files can be found on the web, from plain text and HTML documents to images, videos, music, and much more.

Using web broswers (e.g., Chrome, Firefox, Mozilla), users can make requests on the internet for any one of these various files. The internet sends the request, using an http protocol, to a server. The server, which houses the files, responds to the request with files.

Introduction to HTML

HTML is made up of...
• Text (what you see)
• Markup (what it looks like)
• References to other documents (e.g., images or videos)
